

ATM
=====

### About

* How to collect ATM's GPS coordinates?
  * By using Google Map's Geocoding API (free trial up to $300)
    * Set up billing account (credit card must be enabled for Ecom service)
    * Create project
    * Create API key
    * Enable Geocoding API
